# Microprocessor Peripheral Interface & Stepper Motor Control

This repository contains the design, simulation, and implementation of a microprocessor-based system interfacing with multiple hardware peripherals.

## 🛠️ Project Features
- **Stepper Motor Control:** Precise bidirectional rotation control based on custom input angles (0° to 999°).
- **Matrix Keypad (4x4):** Real-time numeric input parsing and command execution.
- **Display Systems:** Multiplexed data visualization using a 16x2 LCD and 7-Segment displays.
- **Synchronized Clock:** Implementation of a digital clock (seconds) displayed simultaneously on the LCD, 7-Segment, and represented analogously via the stepper motor.

## ⚙️ Hardware & Peripherals
- Microcontroller/Microprocessor
- Stepper Motor (15° step angle)
- 4x4 Matrix Keypad
- 16x2 Character LCD
- 7-Segment Display

## 📁 Repository Structure
- `/Code`: Source code for the microprocessor.
- `/Simulation`: Proteus simulation files and schematics.
- `/Docs`: Lab instructions and project definitions.
